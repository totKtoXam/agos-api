git status - просмотр статуса репозитория

track-файлы файлы отслеживаемые гитом
git add <fileName>
Могут быть измененными и неизмененными

untrack-файлы файлы не отслеживаемые файлы
git rm <--cached/-f> <fileName>
--cached - для того, чтобы удалить файл из отслеживаемых
-f	 - для того, чтобы удалить файл из папки


gitignore - игнорирование определенных файлов
touch .gitignore - создание файла для игнорирования определенных файлов

git add * - добавление всех файлов кроме .gitignore в отслеживаемые файлы
git add . - добалвение всех файлов включая и файл .gitignore в отслеживаемые файлы

git commit -m "Message" - фиксирование изменений с сообщением
Первый commit обычно является "Initial commit"


Выполнять после каждого изменения. ТОЛЬКО ЕСЛИ ВСЁ ВЕРНО РАБОТАЕТ{
git add .
git commit -m "Message"
}

git log - журнал действий commit-ов

git diff - выдает разницу между предыдущим и новоизмененным проектом с последнего commit-а

git branch <name> - создание новой ветки

git checkout <name> - переход на указанную ветку
